## Quick Start

1. Clone the project

```bash
$ git clone https://github.com/fajarhikmal214/simple-nats-monitoring.git
$ cd simple-nats-monitoring
```

2. Copy `.env` from `.env.example`

```bash
$ cp .env.example .env
```

3. Running with Docker

```bash
$ docker-compose up --build
```
